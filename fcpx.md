# Final Cut Pro X

## Shortcuts

https://support.apple.com/guide/final-cut-pro/keyboard-shortcuts-ver90ba5929/mac


- playhead modes
  - `A`: default
  - `T`: trim
  - `B`: blade tool
- workspaces
  - `cmd + 0`: default
  - `ctrl + shift + 2`: color correction


### Clips

- copy/paste effects
  - `cmd + C` & `cmd + shift + V`
- trim clip
  - `option + [`: trim to start
  - `option + ]`: trim to end
