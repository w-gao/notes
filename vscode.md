# vscode

## Keyboard shortcuts

### Everywhere

- `cmd + P`: go to file
- `cmd + shift + P`: show all commands
- `ctrl + \``: open terminal


### In code

- `cmd + shift + .`: show/navigate to symbols in file

- `alt + up/down`: move line up and down
- `alt + shift + up/down`: duplicate line

- `cmd + /`: comment/uncomment code

- multiple cursors
  - `alt + click`
